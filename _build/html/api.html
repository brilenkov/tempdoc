scrapers.common.libs – Various common functions¶

common.libs.auto_queries()¶

Settings for the queries to output Creates a list of queries “length” days into the future for the hotels in the “hotels” array

common.libs.check_table()¶

1 Connects to the database 2 Checks how accurate hotels are based on (words from smaller string in the larger string) / words in the smaller string 3 Choose how accurent you want to check (0 <= like <= 1) 4 Add whatever code you want to do with the results

common.libs.cmp(str1, str2)¶

1 Splits the strings into individual words 2 Makes sure the smaller word is in str1 3 Calculates probability of match by: (words from str1 found in str2) / number of words in str1 4 Rounds off the value and returns it

common.libs.cmp_abbr(str1, str2)¶

1 Finds the smaller string and puts it in str1 2 Counts how many characters from str1 match in str2 in the same order

common.libs.dates(length)¶

Calculates the next “length” days into the future and returns each in a mm/dd/yyyy format

common.libs.find_city(city, state, country, c)¶

Selects city from db or inserts city name if not exists

common.libs.find_name(str)¶

1 Cleans the string and splits it into word groups 2 Finds the largest substring and returns it

common.libs.find_price(str)¶

Finds the price in the string and returns it if found

common.libs.find_star(str)¶

1 Normalizes the string for finding the star rating 2 Finds the star rating in the new string and returns it if found 3 Checks if its a resort and returns -1 if so

common.libs.find_states(input)¶
